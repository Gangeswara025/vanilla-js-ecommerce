## ✨ Features
- **Modern UI/UX**: Clean, premium design inspired by modern high-end retail sites.
- **Standalone Cart System**: Emulates robust platforms (like Flipkart) with a dedicated cart page.
- **Persistent Storage**: Cart items are saved in the browser's `localStorage`, meaning users won't lose their items when reloading pages.
- **Dynamic Updates**: Real-time cart counter in the navigation bar.
- **No Frameworks Needed**: 100% Vanilla JS for fast performance and straightforward understanding.

## 🛠️ Tech Stack
- **HTML5** (Semantics and structure)
- **CSS3** (Flexbox, Grid, Custom Animations, Media Queries)
- **Vanilla JavaScript** (DOM manipulation, event handling, localStorage data syncing)

## 🚀 Getting Started

Since this is a vanilla frontend project, there are no node modules to install or complex build steps!

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Gangeswara025/vanilla-js-ecommerce.git
   ```
2. **Open the project:**
   Simply open the `index.html` file in your preferred web browser. Alternatively, you can use the **Live Server** extension in VS Code for a better development experience.
3. **Explore**:
   - Browse products and click "Add to Cart"
   - Check the cart icon to see real-time quantity updates
   - Click "Cart" to view your saved items, modify quantities, and see the responsive total summary!

## 📁 Project Structure

```text
├── cart.html      # Standalone shopping cart layout page
├── index.html     # Main storefront and product listings
├── script.js      # Cart logic, DOM updates, and localStorage handling
└── style.css      # All combined CSS styles for storefront and cart
```

## 💎 Design Philosophy
The UI relies heavily on modern design principles: **ample whitespace**, **soft shadows**, **rounded components**, and **subtle transitions**. It utilizes the `Inter` font family from Google Fonts to provide an exceptional viewing experience across all devices.
